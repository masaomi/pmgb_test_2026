# PMGB.net 同等サイトの機能要件一覧

## サイト概要

**Petit mignon Grand beau**は、フランス・ブルターニュから日本向けにフレンチアンティーク雑貨を販売するECサイトです。規模は中小ですが、海外発送・独自ルールなど独特な特徴があります。

## 必要機能の全体像

### 1. 商品管理（Product Management）

| 機能 | 詳細 |
|------|------|
| 商品登録 | 商品名・説明・価格・カテゴリ・商品番号（例: K260024）・年代・製造元 |

## PMGBサイト特有の重要機能（見落としやすいポイント）

1. **60分仮押さえシステム** — 1点モノのアンティーク向けに必須。Redis等でタイマー管理
2. **送料後計算フロー** — 注文時に送料確定しない独自フロー（重量依存）
3. **SOLD商品アーカイブ** — 売り切れでも商品ページを残す（信頼性・実績表示）
4. **卸売り顧客の二重価格** — 同じ商品を一般価格と卸価格で管理
5. **海外発送特有の免責説明** — 関税・破損・紛失の説明ページ群
6. **隔週更新サイクル** — 次回更新予定日表示、一括新着追加のワークフロー

## 実装した内容

### デザイン面
- **ウォームブラウン系のカラーパレット**でフレンチアンティークの雰囲気を表現
- レスポンシブ対応（モバイル・タブレット・PC）
- ホバーアニメーション、滑らかなトランジション

### 機能面

| 機能 | 詳細 |
|------|------|
| **カテゴリフィルター** | ボタンクリックで即時絞り込み（アクセサリー・食器・家具など） |
| **商品グリッド** | トップページの実際の50商品データを全て収録 |
| **NEW / SOLD バッジ** | 状態を視覚的に表示、SOLD商品は半透明 |
| **商品詳細モーダル** | カードクリックで詳細ポップアップ（サイズ・重さ・状態・説明） |
| **ショッピングカート** | サイドドロワー形式、`localStorage`で状態を保持 |
| **ページネーション** | 24件ずつ表示 |
| **サイドバー** | 更新スケジュール・為替レート・お支払い・リンク集 |
| **ヘッダー** | スティッキー固定、カートアイコン（件数バッジ付き） |
| **ハンバーガーメニュー** | モバイル時に折りたたみナビ |
| **トースト通知** | カート追加時に画面右下に通知 |

### 元サイトとの違い・注意点
- 商品画像は実際のサイトから取得できないため、カテゴリ絵文字のプレースホルダー表示
- カート→注文手続きは実際のpmgb.netのシステムに委ねる形（alertで案内）
- 外部リンクはすべて本家pmgb.netに繋いでいます
